# Quick Ask

Quick Ask is a browser extension that lets users highlight text inside ChatGPT and get an instant explanation.

## Features

- Highlight any word, phrase, or sentence in ChatGPT
- Click the widget button to request an explanation
- Renders markdown and syntax-colored code blocks
- Supports storing API configuration in the popup

## Installation

1. Clone or download this repository.
2. Open the browser extension page:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. Enable **Developer mode**.
4. Click **Load unpacked**.
5. Select the extension folder.
